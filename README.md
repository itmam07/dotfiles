# My dotfiles
This is a repo for dotfiles for various things like zsh, starship ...

## Current configs:

#### starship
> NOTE: to set the starship config file on windows, you have to add the following line to your $PROFILE
```bash
$ENV:STARSHIP_CONFIG = "$HOME\starship.toml"
```

#### PowerShell7
This script contains some config for git aliases on windows (which are a pain)
